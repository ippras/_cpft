# CPFT

temperature

* ConstantFlow(1mL÷min,150C,10C÷min,Air) - воздух

* ConstantFlow(1mL÷min,150C,9C) - сделана четвертая повторность
* ConstantFlow(1mL÷min,150C,10C) - сделана четвертая повторность
